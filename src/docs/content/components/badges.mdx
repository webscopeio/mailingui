## Installation

If you haven't initialized MailingUI in your project, please refer to [Installation](/docs/installation) docs first.

Choose an a) automatic install by running our `@mailingui/cli` following the commands/options provided or b) install manually by copying and pasting directly into your `src/mailingui/components` directory.
